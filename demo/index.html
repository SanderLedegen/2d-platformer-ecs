Platformer

A 2D platformer written in TypeScript using the ECS (Entity Component System) pattern. The concepts used are greatly inspired by the online course “Intro to C++ Game Programming (COMP4300)” by Dave Churchill of the Memorial University in Newfoundland, Canada.

Thanks to Kenney for the sprites!

Thanks to Craftpix to host the free-to-use tiny hero character sprites!

  • WASD to move;
  • Esc to pause the game;
  • R to reset the main character;
  • B to toggle the collision boxes of every entity;
  • G to toggle the grid (to help building levels);
  • T to toggle the textures.
